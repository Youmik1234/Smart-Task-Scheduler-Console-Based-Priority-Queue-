# Smart Task Scheduler (Java)

This is a Java console-based Task Scheduler that manages tasks based on priority using a Priority Queue.
Features
- Add new tasks
- Assign priority (High, Medium, Low)
- View tasks in sorted order
- Delete highest priority task
- Technologies Used
- Java
- PriorityQueue
- Scanner

How to Run
1. Open TaskScheduler.java in any Java IDE
2. Compile and run the program
3. Use menu to manage tasks

Sample Input
- Task: Complete Java Project → Priority 1
- Task: Study for Exam → Priority 2

Output
Tasks will be displayed in order of priority automatically.
